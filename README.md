# falcond-profiles

All profiles are in /usr/share/falcond/profiles as .conf files

Examples of the format is as follows:

```
name = "cs2"
scx_sched = lavd
scx_sched_props = gaming
performance_mode = true
vcache_mode = cache
```
```
name = "Cyberpunk2077.exe"
scx_sched = none
performance_mode = true
vcache_mode = freq
```
